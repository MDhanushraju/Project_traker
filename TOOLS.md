# Tools Used in Project Tracker

## Backend (Java / Spring Boot — Maven)

| Tool | Purpose |
|------|---------|
| **Maven** | Build & run (mvnw.cmd wrapper; no global mvn needed) |
| **Java 21** | Runtime |
| **Spring Boot 3.2** | Framework |
| **Spring Security** | Authentication & authorization |
| **Spring Data JPA** | Database access |
| **PostgreSQL** | Database |
| **JJWT** | JWT creation & validation |
| **SpringDoc OpenAPI** | Swagger / API docs |
| **JUnit / Spring Test** | Unit tests |
| **Sonar Maven Plugin** | Static analysis (SAST) |

## Frontend (Flutter / Dart)

| Tool | Purpose |
|------|---------|
| **Flutter** | UI framework |
| **Dart (SDK ≥3.0)** | Language |
| **shared_preferences** | Local storage (web) |
| **flutter_secure_storage** | Secure storage (mobile) |
| **Dio** | HTTP client |
| **flutter_lints** | Linting |
| **Flutter Test** | Unit & widget tests |
